Assignment06 jQuery playground

Selectors are used to do exactly what their name implies: select content within a Web page.

They use a very CSS-like syntax in order to do this. To take a few examples:

  • $("tag"): Select all of the <tag> elements shown in red border
  • $(".className"): Select all elements that have the className class applied showed in black border
  • $("#identifier"): Select the element with the id of identifier showed in green border
  • $("tag.className"): Select all <tag> elements that have the className class applied showed in yellow border
